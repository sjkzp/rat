【レース操作方法①】,,アクセル：マウス右ボタン / Zキー<br>クラッチ：マウス左ボタン / Shiftキー
【レース操作方法②】,,シフト操作（マウスの場合）：クラッチを押したままマウスジェスチャー<br>シフト操作（キーボードの場合）：クラッチを押したまま矢印キーで変更

// レーサーセッティング
#racer_setting
WipeIn,AroundToCenter
// CPUの設定
SetRacer,EVR_TYPE_S,1,1.0,3,0.06,0.20,0.60,Standard,MotorCycle5,2.7,0.4
// プレイヤーの設定
SetRacer,Pizza,1,1.0,3,0.06,0.20,0.45,Player,MotorCycle5,2.5,0
Jump,#race

// レース
#race
// 背景の設定
SetBackground,race_sky_night
SetRaceBackground,race_road,race_city
// レースBGMの設定
SetBGM,race
// 距離、勝ったときのタグ、負けたときのタグを設定
RaceSetup,400,#race_win,#race_lose
WipeOut,AroundToCenter
RaceStart

// 勝ったときのタグ
#race_win
WipeIn,LeftToRight
ClearRacer
ClearEffect
ClearItem
SetBackground,bg_A
SetRaceBackground
SetBGM,mute
WipeOut,RightToLeft
SetPortrait,Koichiro,faceKoichiro_B,0.75,0.25
山田コウイチロウ,Koichiro,俺の勝ちだ！！
// キャラ画像・背景をクリア
ClearPortrait
SetBackground,blue
Jump,#retry_select

// 負けたときのタグ
#race_lose
WipeIn,LeftToRight
ClearRacer
ClearEffect
ClearItem
SetBackground
SetBackground,bg_A
SetRaceBackground
SetBGM,mute
WipeOut,RightToLeft
SetPortrait,Miki,faceMiki_B,0.25,0.25
白州ミキ,Miki,私の勝ちだね！

,,選択してください
