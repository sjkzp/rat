RAT custom font

Place one custom font file in this folder.

Recommended file names:
- font.ttf
- font.otf

The browser version automatically loads font.ttf or font.otf when published on a Web server.
When a Mods folder or ZIP is selected, the first .ttf, .otf, .woff, or .woff2 file is used.

The Windows exe version uses font.ttf or font.otf first, then the first .ttf or .otf file in alphabetical order.

Use a font that contains every Japanese character required by the game.
Remove the custom font file to restore the default RAT font.
